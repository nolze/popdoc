Popdoc

Simple Markdown document converter in Node.js

Easy

Set title, language, css, etc. in the YAML front matter

Human-friendly

Preview generated HTML documents as-is live in browser

Customizable

Write your own converter in JavaScript benefiting from the unified/remark ecosystem

# Try it!

# install
npm -g popdoc

# Convert
popdoc example.md -o example.html

# Live preview in browser
popdoc example.md -o example.html --watch
